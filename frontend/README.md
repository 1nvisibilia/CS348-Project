### Frontend Component of CS348 Project

#### Commands
To Run the app in development mode, simply run `npm run dev` in the current folder
You will see the following message:
```
  VITE v4.1.2  ready in 585 ms

  ➜  Local:   http://127.0.0.1:xxxx/
  ➜  Network: use --host to expose
  ➜  press h to show help
```
Go to the url of "Local" to see the app.

Thats's probably all we need right now, feel free to add in any other npm libraries that you see fit.
